# Учеты (офлайн)

Минималистичное офлайн-веб-приложение для учета операций.

## Запуск на ПК (Windows)

Важно: для корректной работы `service worker` нужен локальный сервер (не `file://`).

### Вариант 1: Python

1. Откройте PowerShell в папке проекта `E:\Apps\Uchet`
2. Запустите:

```bash
python -m http.server 8000
```

3. Откройте в браузере:

`http://localhost:8000/`

### Вариант 2: VS Code Live Server

Если используете VS Code — можно открыть папку и запустить расширение Live Server.

## iPhone (Safari) + “На экран Домой”

1. Убедитесь, что iPhone и ПК в одной сети Wi‑Fi.
2. Узнайте IP-адрес ПК (например `192.168.1.10`).
3. Запустите сервер на ПК (см. выше) на порту `8000`.
4. Откройте на iPhone в Safari:

`http://<IP-ПК>:8000/`

5. Нажмите кнопку “Поделиться” в Safari.
6. Выберите “На экран «Домой»”.
7. Откройте приложение с экрана Домой — оно будет работать офлайн после первого открытия.

## Данные

- Данные хранятся в `localStorage`.
- Экспорт/импорт находятся в верхней панели.
- Импорт **заменяет** текущие данные.
